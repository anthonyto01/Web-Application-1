What do we do?

This website allows users to use its functions to find out more about their chosen job.

This can be through the qualifications required and the skill sets needed.

There is a calculator provided which allows the user to find out their payrate they would receive based on the user input of how many hours they work as well as their hourly pay rate.

If the user would like to know about further information or report any queries, there is an about us page which gives the user details about the developer of the website.

This page will have a contact form users can use to voice their queries.
